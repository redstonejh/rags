# RAGS — Build Progress

*Last updated: 2026-06-12. Repo: https://github.com/redstonejh/rags*

## Honest status: systems prototype — NOT yet a playable game

M0–M8 below are still primarily **simulation systems verified by headless, function-level tests**. Phase 0/1 now also has a scripted-input playtest harness that drives the real input → UI → world loop through movement, travel, dialogue, phone, inventory, shop, and pause checkpoints, with windowed screenshot review. The game has still never had a human playtest pass. The visuals are generated programmer-art placeholders rather than final art. Jail/hospital/marriage/elections are button-plus-toast stubs; driving is a speed multiplier; minigames, factions, the radio DJ, build mode, 6 of 12 origins, and the per-origin unique mechanics don't exist yet. Known-risk untested areas: pacing/balance by feel and human long-session play.

What IS solid: the architecture (data-driven defs, record/view separation, EventBus, ironman saves that round-trip everything) and the design's signature mechanics implemented at the math layer.

**The path from here to a functional game is `docs/REBUILD_PLAN.md`** — 3/4 angled perspective, generated pixel art, WASD + click-to-move, and one system at a time taken to interactively-verified 100%.
